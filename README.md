# <p align="center">Binary, Hexadecimal And Decimal Converter</p>

### My Goal -

The aim of this project was to practise Python maths, string validation and formatting techniques.

### Inputs -

| Command | Description |
| --- | --- |
| `convert` | Convert between Binary, Hexadecimal, Decimal |
| `quit` | Exit the program |
| `btd` | Binary to Decimal |
| `bth` | Binary to Hexadecimal |
| `dtb` | Decimal to Binary |
| `dth` | Decimal to Hexadecimal |
| `htb` | Hexadecimal to Binary |
| `htd` | Hexadecimal to Decimal |

### Dependencies -

- [Python](https://www.python.org/downloads/)
